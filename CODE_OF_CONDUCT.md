# Code Of Conduct

## Our Standard

Please keep communication respectful, practical, and focused on improving the project.

Examples of positive behavior:

- assuming good intent
- giving actionable technical feedback
- respecting different experience levels
- keeping discussions focused on the code, documentation, and operator impact

Examples of unacceptable behavior:

- personal attacks
- harassment
- deliberate hostility
- publishing private infrastructure details, credentials, or internal server data without permission

## Enforcement

Project maintainers may remove comments, issues, or contributions that do not follow this code of conduct.
