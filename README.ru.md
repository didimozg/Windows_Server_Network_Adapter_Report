# Windows Server Network Adapter Report

Monorepo с двумя локализованными редакциями одного PowerShell-решения для инвентаризации сетевых адаптеров
Windows Server, default gateway и сохранённых TCP/IP-профилей старых адаптеров.

## Структура репозитория

- [en/](./en/): полная английская редакция
- [ru/](./ru/): полная русская редакция
- [LICENSE](./LICENSE): общая лицензия MIT

## Что входит в каждую редакцию

В каждой редакции есть свои:

- локализованный entry-point скрипт
- локализованный help
- локализованные runtime-сообщения
- документация
- пример файла `servers.txt`

Логика сбора данных в редакциях эквивалентна. Меняется именно язык интерфейса и документации для оператора.

## Зачем такой формат репозитория

Эта структура нужна, чтобы публиковать один Git-репозиторий и не поддерживать два отдельных репозитория только из-за
различий по языку.

Она удобна, если вам нужны:

- один issue tracker
- единый релизный процесс
- одна лицензия
- общая история stars и forks
- две операторские языковые редакции
